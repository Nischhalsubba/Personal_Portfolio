# Repository Instructions

## Repository role

This is an early 2021 static portfolio archive. Preserve it as evidence of design and frontend growth rather than silently rewriting it into the current portfolio.

## Setup

```bash
python -m http.server 8000
```

## Key files

- `index.html`: homepage and project previews.
- `about.html`: early profile content.
- `single.html`: project-detail experiment.
- `dest/style.css`: compiled runtime styling.
- `js/app.js`: navigation and interaction behavior.
- `assets/Images/`: local media.

## Change rules

- Keep the historical year and context visible.
- Replace broken external media with local archival copies where rights allow.
- Label placeholder projects and copy honestly.
- Do not overwrite historical content with current professional claims without documenting the change.
- Preserve the simple static architecture unless a separate modernization branch is created.

## Verification

1. Test all three HTML pages.
2. Check mobile navigation and AnimXYZ loading.
3. Review external fonts, icons, and images.
4. Verify project and social links.
5. Check keyboard access, contrast, reduced motion, and responsive project rows.

## Do not

- Do not present placeholders as shipped client work.
- Do not use this archive as the current canonical portfolio.
- Do not publish images without permission.
- Do not present the local repository thumbnail as a runtime screenshot.
